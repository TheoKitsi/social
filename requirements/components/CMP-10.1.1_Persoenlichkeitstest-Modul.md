# CMP-10.1.1: Personality Test Module

## Functions

| FN-ID | Function |
|---|---|
| **FN-10.1.1.1** | The system shall provide an optional personality test based on the Big Five model (Openness, Conscientiousness, Extraversion, Agreeableness, Neuroticism). |
| **FN-10.1.1.2** | The system shall provide an optional attachment style test based on Attachment Theory (secure, anxious, avoidant, disorganized). |
| **FN-10.1.1.3** | The system shall provide an optional value prioritization test based on the Schwartz model (e.g. security, stimulation, hedonism, achievement, power, universalism, benevolence, tradition, conformity, self-direction). |
| **FN-10.1.1.4** | The system shall display a summarized, understandably presented personality profile to the user after completion. |
| **FN-10.1.1.5** | The system shall allow the user to confirm the computed personality profile or manually adjust individual dimensions. |
| **FN-10.1.1.6** | The system shall feed the test results into the matching algorithm as an additional factor — personality compatibility influences the overall score. |
