# CMP-2.1.1: Funnel Self-Description Module

## Functions

| FN-ID | Function |
|---|---|
| **FN-2.1.1.1** | The system shall provide the following self-description fields for Level 1 (Mandatory): basic data (age, gender, location), relationship intention, core values, sexual orientation, languages. |
| **FN-2.1.1.2** | The system shall provide the following self-description fields for Level 2 (Mandatory): personality type, introversion/extraversion, communication style, daily routine, diet, smoking/alcohol/drugs, pets, religion/spirituality, political orientation. |
| **FN-2.1.1.3** | The system shall provide the following self-description fields for Level 3 (Mandatory): desire for children + timeline, career ambitions, income situation, housing goal, willingness to relocate, educational background, family vision, retirement planning attitude. |
| **FN-2.1.1.4** | The system shall provide the following self-description fields for Level 4 (Optional): hobbies (detailed), sports/fitness, travel preferences, music taste, sexual preferences/openness, quirks/fetishes, unconventional life models, special family bonds, fears/phobias (partner-relevant). |
| **FN-2.1.1.5** | The system shall provide the following self-description fields for Level 5 (Optional): food preferences (detailed), sleep habits, tidiness/chaos tolerance, social media usage, media consumption, cultural traditions, holidays, humor type, conflict resolution style. |
| **FN-2.1.1.6** | The system shall enforce sequential level unlocking — Level N+1 becomes accessible only after Level N is fully completed. |
