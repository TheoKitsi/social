# CMP-8.1.1: Reflection Onboarding Assistant

## Functions

| FN-ID | Function |
|---|---|
| **FN-8.1.1.1** | The system shall provide a multi-step, conversation-based onboarding assistant that guides the user through all 5 funnel levels — starting with "Who am I?" followed by "Who am I looking for?" per level. |
| **FN-8.1.1.2** | The system shall display a progress bar showing the overall progress across all onboarding steps and funnel levels. |
| **FN-8.1.1.3** | The system shall pose at least one reflection question before each topic block, encouraging the user toward honest self-reflection (e.g. "What if your partner never wanted to exercise? How would that feel for you?"). |
| **FN-8.1.1.4** | The system shall provide a context explanation for each profile field, explaining why this field is relevant and what influence it has on match quality. |
| **FN-8.1.1.5** | The system shall enforce a deliberate pace in onboarding — no gamification elements, no time pressure, no reward mechanics, no comparisons with other users. |
| **FN-8.1.1.6** | The system shall automatically save the intermediate state on interruption and invite the user to continue on next login, showing the last editing point. |
