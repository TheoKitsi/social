# SOL-8: Guided Reflective Onboarding Assistant

| Metric | Value |
|---|---|
| User Stories | 3 |
| Components | 3 |
| Functions | 12 |

The onboarding is not a form — it is a **reflection process**. A conversational assistant guides the user step by step through the 5 funnel levels, with reflection questions, context explanations, and a personalized life analogy. Deliberate pacing instead of rush.

---

## User Stories

- US-8.1: Guided Initial Profile Population via Reflection Assistant
- US-8.2: Personalized Analogy in Onboarding
- US-8.3: Later Completion of Optional Fields
