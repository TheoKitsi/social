# US-17.1: Feedback bei Kontolöschung

> **Als** Plattformbetreiber **möchte ich** beim Verlassen eines Nutzers optionales Feedback einholen, **damit** die Plattform kontinuierlich verbessert werden kann und die Erfolgsrate messbar wird.

## Akzeptanzkriterien

- [ ] AC-17.1.1: Bei Kontolöschung wird ein optionales Exit-Interview mit vordefinierten Fragen angezeigt.
- [ ] AC-17.1.2: Die Kernfrage „Haben Sie über diese Plattform einen Partner gefunden?" wird prominent gestellt.
- [ ] AC-17.1.3: Verbesserungsvorschläge können als Freitext eingegeben werden.
- [ ] AC-17.1.4: Alle Exit-Daten werden anonymisiert gespeichert und ausschließlich für Produktanalyse verwendet.
- [ ] AC-17.1.5: Bei expliziter Zustimmung können Erfolgsgeschichten anonymisiert für Marketing freigegeben werden.
- [ ] AC-17.1.6: Das Exit-Interview ist vollständig optional — Kontolöschung ist auch ohne Teilnahme möglich.
- [ ] AC-17.1.7: Die Kontolöschung wird durch das Exit-Interview nicht verzögert oder erschwert.
