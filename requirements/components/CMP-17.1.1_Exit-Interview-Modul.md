# CMP-17.1.1: Exit Interview Module

## Functions

| FN-ID | Function |
|---|---|
| **FN-17.1.1.1** | The system shall offer an optional exit interview with predefined questions during account deletion: reason for deletion, platform rating (1-5 stars), and open improvement suggestions. |
| **FN-17.1.1.2** | The system shall separately capture whether the user found a partner (Yes / No / Prefer not to say) as the central success metric. |
| **FN-17.1.1.3** | The system shall offer a free-text field for improvement suggestions within the exit interview. |
| **FN-17.1.1.4** | The system shall store all exit interview data anonymized and use it exclusively for aggregated product analysis. |
| **FN-17.1.1.5** | The system shall offer optional opt-in for an anonymized success story --- usable for marketing purposes only with explicit consent. |
