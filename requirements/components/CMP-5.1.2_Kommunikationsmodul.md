# CMP-5.1.2: Communication Module

## Functions

| FN-ID | Function |
|---|---|
| **FN-5.1.2.1** | The system shall provide an end-to-end encrypted chat after mutual reveal. |
| **FN-5.1.2.2** | The system shall provide a report function through which users can report inappropriate behavior; reports are forwarded to the moderation team. (Technical implementation of the reporting system see SOL-7, FN-7.1.2.4) |
| **FN-5.1.2.3** | The system shall provide a block function that immediately prevents all contact and permanently removes the blocked profile from the blocking user's matching list. |
