# CMP-2.2.1: Funnel Target Person Module

## Functions

| FN-ID | Function |
|---|---|
| **FN-2.2.1.1** | The system shall provide the same 5-level field structure for the target person description as for the self-description. |
| **FN-2.2.1.2** | The system shall capture a weighting (Must-Have / Nice-to-Have / Indifferent) for each target person field. |
| **FN-2.2.1.3** | The system shall capture tolerance ranges (exact / range / flexible) for numeric and categorical target person fields. |
| **FN-2.2.1.4** | The system shall provide a deal-breaker flag per target person field that triggers immediate matching exclusion if not met. |
