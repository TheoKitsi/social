# CMP-8.2.1: Analogy System

## Functions

| FN-ID | Function |
|---|---|
| **FN-8.2.1.1** | The system shall offer a selection of personalized analogies at the beginning of onboarding (at least: apartment search, car purchase, smartphone, travel, clothing) as well as the option "Without analogy". |
| **FN-8.2.1.2** | The system shall embed the chosen analogy into explanatory texts and headings per funnel level to make the process tangible. |
| **FN-8.2.1.3** | The system shall allow the user to change or deselect the analogy at any time. |
